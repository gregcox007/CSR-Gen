# CSR-Gen
Bash Script to Generate CSRs

PreReq: OpenSSL must be installed

Note: Tested on Mac, may not work on any other OS

Instructions:
1. Create a Folder
2. Download csr.sh Script to newly created folder
3. Open Terminal and cd to your newly created folder
4. Run the command "chmod 777 csr.sh"
5. Then run the command "./csr.sh"
6. Input the required prompts
7. Done send .csr file to you customer

This script creates three files
1. A config file (san.cnf.org)
2. A .csr file
3. A .key file
